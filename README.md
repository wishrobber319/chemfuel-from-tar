# Chemfuel from Tar

Reworks Vanilla Factions Expanded - Medieval 2's alchemy-workbench chemfuel recipes for a Medieval Overhaul tar economy:
- Removes "Make chemfuel from organics".
- Changes "Make chemfuel from wood" into "Make chemfuel from tar", consuming Medieval Overhaul's tar (DankPyon_Tar, gathered at the mine shaft) instead of wood logs. Amounts are unchanged (70 tar -> 25 chemfuel).

Requires both Vanilla Factions Expanded - Medieval 2 (the recipes) and Medieval Overhaul (the tar item); does nothing without both.
